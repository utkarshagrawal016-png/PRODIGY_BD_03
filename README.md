# PRODIGY_BD_03 - JWT Authentication System

A secure authentication and authorization system built using Flask, MongoDB, JWT, and bcrypt.

## Features
- User Registration & Login
- Password Hashing with bcrypt
- JWT Token Generation
- Protected Routes
- Role-Based Access Control
- Separate User/Admin Dashboards
- Dedicated Profile Pages
- Modern Responsive Frontend UI

## Tech Stack
- Python
- Flask
- MongoDB
- Flask-JWT-Extended
- bcrypt
- HTML/CSS/JavaScript

## Run Locally
1. Clone repo
2. Install requirements
3. Configure `.env`
4. Run `python app.py`

## Learning Outcomes
- Authentication vs Authorization
- JWT Workflow
- Role-Based Access Control
- Secure Password Storage
- Frontend/Backend Integration
